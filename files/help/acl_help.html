Object Permission Description
acl add Allow user to add acl items
edit Allow user to edit acl items
delete Allow user to delete acl items
view Allow user to view acl items
cfg edit Allow user to edit configurations
view Allow user to view configurations
file_<FILE>
file_<DIR> 		add Allow user to create new files
edit Allow user to edit files
delete Allow user to delete files and folders
browse Allow user to browse folder
rename Allow user to rename files and folders
mkdir Allow user to create folders
upload Allow user to upload files
download Allow user to download files
log view Allow user to view log
clear Allow user to clear log
post
post_<ID> 		add Allow user to add post
view Allow user to view post
edit Allow user to edit post
modify Give user limited right to edit own posts
delete Allow user to delete post
remove Give user limited right to remove own posts
reply Allow user to reply post (tree limited)
user register Allow user to register new users
edit Allow user to edit users
delete Allow user to delete
view Allow user to view user profile
view_sessions Allow user to view active sessions
profile Allow user to edit himself
file view Allow user to preview files
